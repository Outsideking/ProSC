# ProSC

โปรแกรมสำเร็จรูปสำหรับใช้งานเชื่อมต่อกับ **Scanzaclip** และโมดูลอื่นๆ ที่สามารถเพิ่มได้ในอนาคต

## Features
- เชื่อมต่อ Scanzaclip เพื่อจัดการข้อมูลและสั่งงาน
- โมดูลแบบแยกโฟลเดอร์ สามารถเพิ่มโมดูลใหม่ได้ง่าย
- มีไลบรารีช่วยเหลือ common functions สำหรับทุกโมดูล
- รองรับ Python environment พร้อม dependencies ผ่าน `requirements.txt`

## Installation
```bash
git clone https://github.com/username/ProSC.git
cd ProSC
pip install -r requirements.txt
